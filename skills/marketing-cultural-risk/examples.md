# 검토 예시

이 파일은 SKILL.md의 검토 기준을 실제 사례로 설명한다.

---

## 예시 1: Critical — 역사적 날짜와 표현이 결합된 경우

| 항목 | 내용 |
|------|------|
| 시장 | 한국 |
| 날짜 | 5월 18일 |
| 캠페인명 | Tank Day |
| 제품명 | Tank Tumbler |
| 카피 | 책상에 탁! |
| 기대 리스크 | **Critical** |

### 왜 위험한가

이 경우 문제는 단어 하나가 아니다.

날짜, 캠페인명, 제품명, 카피가 결합되면서 한국 사회의 역사적 상처와 국가폭력 맥락을 연상시킬 수 있다.

- "5월 18일"은 광주 민주화운동과 직접 연결된다.
- "Tank"는 군사 장비를 연상시킨다.
- "탁!"은 물리적 충격이나 폭력을 떠올리게 할 수 있다.

이 조합은 의도와 관계없이 역사적 상처를 희화화하는 것으로 해석될 가능성이 높다.

**최종 권고: 즉시 중단 및 검토**

> 이 예시는 특정 브랜드를 공격하기 위한 것이 아니라, 조합 리스크가 어떻게 작동하는지를 보여주는 패턴이다.

## 예시 2: Medium 또는 High — 폭력적 은유

| 항목 | 내용 |
|------|------|
| 캠페인명 | Bomb Sale |
| 카피 | 가격 폭격 |
| 기대 리스크 | **Medium ~ High** |

### 왜 위험한가

평상시에는 흔한 과장 표현으로 보일 수 있다.

하지만 전쟁·테러·사고·참사 관련 뉴스가 있는 시기에는 부정적으로 해석될 수 있다.

- "Bomb"은 폭발·테러를 연상시킨다.
- "폭격"은 군사 행동을 떠올리게 한다.
- 실시간 이슈 검토 결과에 따라 High로 상향될 수 있다.

**실시간 이슈가 없으면 Medium, 있으면 High**

## 예시 3: Low — 중립적 캠페인

| 항목 | 내용 |
|------|------|
| 캠페인명 | Summer Tumbler Week |
| 카피 | 여름을 더 시원하게 |
| 기대 리스크 | **Low** |

### 왜 안전한가

- 역사적 날짜와 관련 없음
- 폭력적 표현 없음
- 피해자 관련 표현 없음
- 조롱 요소 없음
- 조합 리스크 없음

**최종 권고: 진행 가능**

## 예시 4: 실시간 이슈로 리스크가 상향되는 경우

| 항목 | 내용 |
|------|------|
| 캠페인명 | Crash Price Day |
| 카피 | 가격 대폭락 |
| 출시일 | 대형 사고 발생 당일 |
| 기본 리스크 | **Medium** |
| 실시간 이슈 반영 후 | **High** |

### 왜 상향되는가

표현 자체는 가격 할인 은유일 수 있다.

하지만 당일 대형 사고나 사망 사고가 발생한 경우 "Crash", "폭락" 같은 표현이 부적절하게 받아들일 수 있다.

- 소비자가 슬픔이나 충격 상태일 때 가벼운 표현은 무례하게 느껴진다.
- 언론이 "브랜드가 사고를 마케팅에 이용"이라고 보도할 수 있다.

**이 예시는 실시간 이슈 검토가 왜 필요한지를 보여준다.**
