# 실시간 이슈 검토 워크플로우

출시 직전 실시간 사회적 이슈와의 충돌 가능성을 검토하는 워크플로우다.

---

## 고정 리스크 vs 실시간 리스크

### 고정 리스크

스킬 파일 기준으로 언제든 검토할 수 있는 리스크:

- 역사적 기념일
- 추모일
- 참사일
- 민감 표현
- 조합 리스크

### 실시간 리스크

스킬 파일만으로는 알 수 없고, 검색 도구가 필요:

- 오늘/내일 발생한 사고나 참사
- 갑자기 논란이 된 표현
- SNS에서 확산 중인 밈이나 조롱
- 정치/사회적 갈등
- 지역 이슈
- 브랜드/인물 관련 실시간 논란

> **이 스킬팩 자체는 실시간 데이터를 직접 조회하지 않는다.**
> 웹 검색, 뉴스 검색, SNS 검색, 트렌드 검색 도구가 있는 에이전트 환경에서만 실시간 검토가 가능하다.

---

## 검색해야 할 키워드

| 키워드 유형 | 예시 |
|-------------|------|
| 브랜드명 | 브랜드 이름 |
| 캠페인명 | 캠페인 이름 |
| 제품명 | 제품 이름 |
| 메인 카피 핵심 단어 | 카피에서 핵심이 되는 단어 |
| 해시태그 | 캠페인 해시태그 |
| 출시일 | 출시 날짜 |
| 출시 지역 | 출시 지역 |
| 사회 이슈 키워드 | "논란", "사고", "참사", "불매", "비판", "사과" |

---

## 출력 형식

```md
# 실시간 이슈 검토 결과

## 검색 가능 여부
가능 / 불가능

## 확인한 키워드
-
-

## 발견된 실시간 이슈
-
-

## 캠페인과의 충돌 가능성
Low / Medium / High / Critical

## 리스크 상향 여부
상향 없음 / Medium으로 상향 / High로 상향 / Critical로 상향

## 권고
진행 / 수정 / 보류 / 중단
```

## 검색 도구가 없는 경우

결과에 아래 문장을 명시한다:

> "실시간 이슈 검토는 수행하지 못했으며, 고정 리스크 기준으로만 판단했습니다."

이 경우 캠페인 담당자가 직접 뉴스·SNS를 확인한 후 출시를 진행해야 한다.
