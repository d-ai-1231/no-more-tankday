# no-more-tankday

**한국 시장용 마케팅 콘텐츠의 문화·역사·사회적 리스크를 출시 전에 검수하는 AI 에이전트 스킬팩**

---

## 이 프로젝트가 해결하는 문제

마케팅 캠페인은 의도가 좋아도, 특정 날짜·표현·제품명·비주얼·사회적 맥락이 결합되면 대중에게 전혀 다르게 해석될 수 있다.

이 스킬팩은 캠페인 공개 전에 그런 리스크를 발견하고, High 이상이면 출시 전 수정하도록 돕는다.

**이 프로젝트는 특정 브랜드를 공격하기 위한 것이 아니다.** 누구나 반복되는 실수를 예방할 수 있도록 만든 범용 검수 도구다.

---

## 이 스킬팩이 검토하는 것

| 검토 항목 | 설명 |
|-----------|------|
| 캠페인명 | 브랜드 캠페인 이름 |
| 제품명 | 출시 제품 이름 |
| 출시일 | 제품 공개 날짜 |
| 이벤트 날짜 | 프로모션·팝업 등 이벤트 날짜 |
| 프로모션 기간 | 할인·이벤트 진행 기간 |
| 메인 카피 | 핵심 광고 문구 |
| 서브 카피 | 보조 문구 |
| 해시태그 | SNS 해시태그 |
| 비주얼 설명 | 이미지·영상·일러스트 묘사 |
| 콘티 흐름 | 영상·캠페인 스토리보드 |
| 한국 사회·역사·문화 맥락 | 날짜·표현의 사회적 의미 |
| 실시간 사회적 이슈 | 오늘/내일 발생한 이슈와의 충돌 |

---

## 중요한 개념: 고정 리스크 vs 실시간 리스크

### 고정 리스크

스킬 파일이 기준으로 검토할 수 있는 리스크다.

- 역사적 기념일, 추모일, 참사일, 민주화운동 관련 날짜
- 폭력·군사·국가폭력·차별·조롱 연상 표현
- 날짜 + 캠페인명 + 카피 조합 리스크

### 실시간 리스크

스킬 파일만으로는 알 수 없고, 에이전트가 웹 검색·뉴스 검색·SNS 검색·트렌드 검색 도구를 사용할 수 있을 때 확인할 수 있는 리스크다.

- 오늘 발생한 참사나 사고
- 갑자기 논란이 된 단어
- 특정 지역·인물·브랜드 관련 실시간 이슈
- SNS에서 확산 중인 비판 여론
- 내일 예정된 정치·사회적 이벤트

> **이 스킬팩 자체는 실시간 뉴스를 직접 조회하지 않는다.**
> 실시간 이슈 검토가 필요한 경우, 사용 중인 에이전트가 웹 검색·뉴스 검색·SNS 모니터링·트렌드 검색 도구에 접근할 수 있어야 한다.

## 기본 워크플로우

```
캠페인 생성
→ 카피/비주얼/콘티 생성
→ 고정 문화·역사 리스크 검수
→ 가능한 경우 실시간 이슈 검토
→ High 이상이면 수정
→ 재검수
→ 최종 출력
```

---

## 리스크 등급

| 등급 | 의미 | 처리 방식 |
|------|------|-----------|
| **Low** | 의미 있는 리스크 없음 | 진행 가능 |
| **Medium** | 오해 가능성 있음 | 수정 권장 |
| **High** | 공개 비판 가능성 높음 | 수정 전 출시 금지 |
| **Critical** | 불매·언론 확산·피해자 커뮤니티 반발 가능성 | 출시 중단 및 재검토 |

## 면책 문구

이 도구는 법무, PR, 문화적 안전성을 보장하지 않는다.
출시 전 리스크 검토를 돕는 보조 도구이며, 중요한 캠페인은 반드시 사람의 검토와 함께 사용해야 한다.
