# no-more-tankday

### 출시 전에 터지는 걸 막는 AI 마케팅 검수 스킬팩

> 캠페인은 의도대로 만들어지지만, 해석은 대중의 몫이다.
> 이 스킬팩은 대중이 화내기 전에 먼저 발견하게 해준다.

---

## 왜 이걸 만났나요?

마케터의 의도는 항상 좋다.
하지만 "괜찮을 거야"라고 생각한 그 캠페인이, 특정 날짜에 특정 표현이 만나면서 전혀 다르게 읽힐 수 있다.

이 스킬팩은 **콘텐츠를 만들어주는 도구가 아니다.**
만들어진 콘텐츠가 한국 사회의 역사·문화·사회적 맥락과 충돌하는지 **출시 전에 검수하는 QA 게이트**다.

---

## 뭘 검토하나요?

캠페인 하나를 통째로 본다.

| 보는 것 | 왜 보는가 |
|---------|----------|
| 캠페인명 | 이름 자체가 논란이 될 수 있다 |
| 제품명 | 특정 사건·집단을 연상시킬 수 있다 |
| 출시일 | 역사적 민감일과 겹칠 수 있다 |
| 카피 | 유명 구호·참사 표현과 닮을 수 있다 |
| 비주얼 | 날짜·카피와 합쳐지면 의미가 바뀐다 |
| 해시태그 | SNS에서 확산 속도가 다르다 |
| 콘티 흐름 | 전체 맥락에서 부적절한 서사가 보일 수 있다 |

**핵심은 조합이다.** 단어 하나는 안전해도, 날짜+이름+카피가 만나면 위험해진다.

---

## 고정 리스크 vs 실시간 리스크

### 고정 리스크 (스킬 파일로 검토 가능)

- 역사적 기념일, 추모일, 참사일
- 폭력·군사·국가폭력·차별·조롱 연상 표현
- 날짜 + 캠페인명 + 카피 조합

### 실시간 리스크 (검색 도구 필요)

- 오늘 아침에 터진 사고나 참사
- 갑자기 논란이 된 단어
- SNS에서 확산 중인 비판 여론

> 이 스킬팩은 실시간 뉴스를 직접 조회하지 않는다.
> 검색 도구(Tavily, Brave Search 등)가 연결된 에이전트 환경에서만 실시간 검토가 가능하다.

| 검색 도구 | 특징 |
|-----------|------|
| Tavily API | 웹+뉴스 통합 검색, 추천 |
| Brave Search API | 월 2,000회 무료 |
| 웹 검색 (범용) | Naver/Daum 뉴스 직접 검색 |

## 설치 (30초)

```bash
git clone https://github.com/d-ai-1231/no-more-tankday.git
cp -r no-more-tankday/skills/marketing-cultural-risk ./skills/
```

## 사용 (복붙)

에이전트 프롬프트에 이 파일 내용을 붙여넣기만 하면 된다:

```
agents/final_qa_prompt.md
```

자세한 사용법은 [integration_guide.md](docs/integration_guide.md)에서 3가지 모드(수동 검토 / QA 게이트 / 생성→검수→수정 루프)를 확인할 수 있다.

---

## 워크플로우

```
캠페인 생성
  → 카피/비주얼/콘티 생성
  → 고정 리스크 검수 (자동)
  → 실시간 이슈 검토 (검색 도구 있을 때)
  → High 이상이면 수정 후 재검수
  → 최종 출력
```

---

## 리스크 등급

| 등급 | 뜻 | 해야 할 일 |
|------|-----|-----------|
| **Low** | 문제 없음 | 그대로 진행 |
| **Medium** | 오해 가능 | 수정 권장 |
| **High** | 비판 예상 | 출시 금지, 수정 후 재검수 |
| **Critical** | 불매·언론 보도 가능 | 출시 중단, PR/법무/임원 검토 |

---

## 면책

이 도구는 법무도, PR도, 문화적 안전성도 보장하지 않는다.
출시 전 리스크를 발견하게 도와주는 보조 도구일 뿐이다.
중요한 캠페인은 반드시 사람의 최종 검토가 필요하다.
