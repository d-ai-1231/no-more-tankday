# 컨셉 플래너 에이전트 프롬프트 애드온

캠페인명과 컨셉명을 정할 때 아래 기준을 참고한다.

---

## 핵심 원칙

- 이름 자체보다 **조합 리스크**를 더 중요하게 본다.
  - 같은 이름이라도 출시일, 지역, 사회적 맥락에 따라 위험도가 달라진다.
- 군사, 폭력, 참사, 조롱, 정치적 구호를 연상시키는 컨셉명은 피한다.

---

## 주의해야 할 컨셉명 패턴

| 패턴 | 예시 | 위험 요소 |
|------|------|-----------|
| 군사 관련 | Tank, Bomb, Attack, Strike, Battle | 폭력·전쟁 연상 |
| 폭력 관련 | Smash, Crush, Kill, Destroy | 물리적 폭력 연상 |
| 점령·정복 | Conquer, Dominate, Occupy | 식민·군사 연상 |
| 정치 구호 | Freedom, Revolution, Liberation | 정치적 해석 가능 |
| 참사 연상 | Crash, Collapse, Blast | 사고·재해 연상 |

---

## 조합 검토 체크리스트

- [ ] 캠페인명 + 출시일 → 역사적 민감일과 겹치는가?
- [ ] 캠페인명 + 제품명 → 부적절한 의미가 만들어지는가?
- [ ] 컨셉명 + 카피 → 폭력·차별·조롱을 강화하는가?
- [ ] 컨셉명 + 비주얼 → 부정적 이미지를 만드는가?
