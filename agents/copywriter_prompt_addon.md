# 카피라이터 에이전트 프롬프트 애드온

카피 작성 시 아래 기준을 예방적 작성 가이드로 참고한다.

---

## 주의해야 할 표현

역사적 상처, 참사, 폭력, 국가폭력, 정치 갈등, 차별, 조롱을 연상시킬 수 있는 표현을 피한다.

### 할인·이벤트 카피에서 특히 주의할 단어

| 단어 | 위험 요소 |
|------|-----------|
| 폭격 | 군사 행동 연상 |
| 박살 | 물리적 폭력 연상 |
| 점령 | 군사·식민 연상 |
| 진압 | 시위 진압·국가폭력 연상 |
| 대폭락 | 사고·사망 뉴스와 결합 시 부적절 |
| 충격 | 참사·재해 뉴스와 결합 시 부적절 |
| Bomb | 폭발·테러 연상 |
| Crash | 사고·충돌 연상 |
| Kill | 죽음·폭력 연상 |

### 조합 주의

표현 자체보다 날짜, 지역, 사회적 맥락과의 조합이 더 중요하다.

예: "폭격"이라는 단어가 전쟁 기념일이나 참사 발생일에 사용되면 리스크가 급상승한다.
